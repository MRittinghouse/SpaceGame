# Ship Upgrades

## Overview
Ships have 3 upgrade slots. Buy upgrades at the Shipyard (accessible from galaxy map).

## Upgrade Catalog

| Upgrade | Price | Slot | Effect |
|---------|-------|------|--------|
| Cargo Bay Extension | 5,000 CR | cargo | +20 cargo capacity |
| Fuel Tank Upgrade | 3,000 CR | fuel | +30 fuel capacity |
| Efficient Engines | 8,000 CR | engine | -2 fuel per jump |
| Mining Drill Mk2 | 6,000 CR | mining | -20% drill time |
| Advanced Scanner | 7,000 CR | scanner | +2 scan charges |

## Rules
- Maximum 3 upgrades installed at once
- No duplicate upgrades
- Uninstalling refunds 50% of purchase price
- Upgrade bonuses stack with skill bonuses
